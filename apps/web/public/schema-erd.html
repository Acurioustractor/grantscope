Data Pipeline — 48 Agents, 9 Categories

CivicGraph's data pipeline is fully automated — 48 agents that ingest from government registries, resolve entities across datasets, enrich with geographic and demographic data, analyse for power concentration and funding gaps, and monitor data quality. Each agent logs runs, items found, and errors to the agent_runs table for full observability.
