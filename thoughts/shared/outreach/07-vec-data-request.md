## Subject

Research data access — Victorian political donation disclosures

## Body

Hello,

I run CivicGraph (https://civicgraph.com.au), a civic infrastructure project
that resolves Australian public entities across procurement, donations, and
charity registers. Publishing investigations on accountability and public-
money flows — examples at https://civicgraph.com.au/reports.

I've been using your public disclosure system at
https://disclosures.vec.vic.gov.au/donations-public/ (when it's out of
maintenance) but scraping a dynamic search interface isn't robust. I'd like
to request a **bulk export** of Victorian political donation disclosures
from 2018 (when real-time disclosure started) to present.

**Fields we'd value:**
- Donor name / ABN / address (where public)
- Donation amount, date
- Recipient party / nominated entity / campaigner
- Disclosure year
- Any provided classification (associated entity, union-affiliated, etc.)

**Preferred format:** CSV or JSON.

**Use:** we cross-reference with federal AEC donation data (already in our
atlas) and government procurement records to surface patterns like
donor-contractor overlap. All usage is attributed, non-commercial, and we
happily share findings back.

If you publish a machine-readable export already — apologies for missing
it; please point me to the URL.

Ben Knight
A Curious Tractor / CivicGraph
ben@benjamink.com.au
