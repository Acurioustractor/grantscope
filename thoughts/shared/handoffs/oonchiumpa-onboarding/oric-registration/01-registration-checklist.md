## Pre-Registration (Week 1 — THIS WEEK)

- [ ] **Call Aboriginal Investment NT** (1800 943 039)
  - Ask: "Can an Aboriginal-controlled Pty Ltd apply for CIIG Round 2, or must we be a registered not-for-profit?"
  - Ask: "If we register an ORIC corporation before 30 April, can we apply as that new entity?"
  - Ask: "Is an auspice arrangement acceptable if our ORIC registration is still processing?"
  - Note the name of who you speak to and their answer

- [ ] **Call ORIC Alice Springs** (1800 622 431)
  - Ask: "We need to register a new small Aboriginal corporation. What's the current processing time?"
  - Ask: "Can we book a pre-incorporation meeting this week or next?"
  - Ask: "Can you email us the standard rule book template and application form?"
  - Request a **pre-incorporation doorway workshop** (email: training@oric.gov.au)

- [ ] **Confirm 5 foundation members** (all must be 15+, majority Aboriginal/TSI)
  - See proposed member list below
  - Each member needs: full name, date of birth, residential address, signature

- [ ] **Confirm 3+ directors** (majority must be Aboriginal/TSI, all must be 18+)
  - See proposed director list below
  - Directors are automatically members

## Registration Application (Week 2)

- [ ] **Choose corporation name** — "Oonchiumpa Aboriginal Corporation" (check availability with ORIC)
- [ ] **Complete rule book** — use ORIC standard template (see `02-draft-rule-book.md`)
- [ ] **Obtain 75% member consent** — members sign consent form agreeing to register and adopt the rule book
- [ ] **Complete application form** — online at oric.gov.au or PDF
- [ ] **Submit to ORIC** — online preferred (faster processing)

## After Submission (Weeks 3-4)

- [ ] ORIC checks application completeness (~14 working days)
- [ ] If incomplete, ORIC writes to request additional info — respond ASAP
- [ ] Corporation registered — ORIC issues ICN (Indigenous Corporation Number)
- [ ] **Immediately lodge CIIG application** using new ORIC entity

## Fallback Plan (if ORIC processing takes too long)

- [ ] **Option A**: Start CIIG eligibility application NOW using existing Pty Ltd, swap entity if/when ORIC registration completes
- [ ] **Option B**: Arrange auspice with existing Alice Springs NFP (Tangentyere Council, CASSE, Congress) — they apply on Oonchiumpa's behalf
- [ ] **Option C**: Register as NT Incorporated Association (~10 working days, ~$100-200) via Licensing NT as faster alternative

## Corporation Size Classification

Oonchiumpa will register as a **SMALL** corporation:
- Revenue < $100,000 per year (at start)
- Gross assets < $100,000
- Fewer than 5 employees

**What this means:**
- Lightest compliance burden
- General report (not full audit) each year
- No requirement for registered auditor
- Can prepare own financial statements
- Still must hold AGM and lodge annual reports with ORIC
